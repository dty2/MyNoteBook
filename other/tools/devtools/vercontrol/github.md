## 使用
* SSH Key
  ```
  ssh-keygen -t rsa -C "邮箱"
  cd ~/.ssh
  cat id_rsa.pub
  ```
复制cat的运行结果到Github中的setting的ssh选项中即可通过SSH连上Github
